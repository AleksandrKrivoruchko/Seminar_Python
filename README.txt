Игра "конфетки"

Что делаем:
1. "fork" репозитория на GitHub
2. "clone" себе на компьютер
3. создаем новую ветку "dev" и работаем в ней
4. делаем коммит(-ы)
5. "push" своей версии репозитория себе на GitHub
6. на GitHub делаем "Pull request" (либо можно сразу из PyCharm)
7. пробуем собрать все в кучу :-)


Используемые методы:
1. draw_candies(count):
    выводит в консоль фразу "Осталось конфет: {}",
    выводит картинки конфет, соответствующие количеству, по 10 в ряду (emojize(':candy:'))
    ничего не возвращает

2. take_input(player_num, min_num, max_num, num_of_cand):
    выводит в консоли вопрос "Игрок {}, сколько конфет возьмешь?"
    принимает ввод от игрока
    проверяет ввод на корректность (чтобы было число, не меньше минимального
    и не больше максимального, и не больше количества оставшихся конфет)
    возвращает число взятых конфет

3. computer_turn(min_num, max_num, num_of_cand):
    просчитывает лучший ход (остаток от деления количества конфет на (максимальное + 1))
    если лучший ход невозможен (остаток от деления равен 0), то делает случайный ход
    выводит в консоль "Компьютер взял конфет: {}."
    возвращает число взятых конфет

4. player_vs_player(min_take, max_take, number_of_candies):
    моделирует игру "игрок против игрока"
    вызывает метод draw_candies
    пока количество конфет больше 0, дает игрокам возможность ходить по очереди -
        вызывает метод take_input
    проверяет условие победы (осталось 0 конфет)
    выявляет победителя и выводит сообщение в консоль

5. player_vs_computer(min_take, max_take, number_of_candies):
    моделирует игру "игрок против компьютера"
    вызывает метод draw_candies
    спрашивает очередность ходов (кто первый)
    пока количество конфет больше 0, игрок и компьютер ходят по очереди -
        вызывает метод take_input
        вызывает метод computer_turn
    проверяет условие победы (осталось 0 конфет)
    выявляет победителя и выводит сообщение в консоль

6. main(min_take=1, max_take=3, number_of_candies=10):
    основной метод, спрашивает вид игры (с компьютером или друг с другом)
    запускает игру -
        вызывает метод player_vs_player или player_vs_computer
    доп опции: ввести количество конфет, и сколько можно брать за один раз
