Калькулятор

Что делаем:
1. "fork" репозитория на GitHub
2. "clone" себе на компьютер
3. создаем новую ветку "dev" и работаем в ней
4. делаем коммит(-ы)
5. "push" своей версии репозитория себе на GitHub
6. на GitHub делаем "Pull request" (либо можно сразу из PyCharm)
7. пробуем собрать все в кучу :-)
